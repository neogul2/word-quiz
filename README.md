# Word Quiz

단어 학습을 위한 퀴즈 애플리케이션입니다.

## 데모

[https://word-quiz-[your-url].vercel.app](https://word-quiz-[your-url].vercel.app)

## 주요 기능

- 영단어 퀴즈
- 학습 진도 관리
- 오답 노트 기능

## 시작하기

개발 서버 실행:

```bash
npm run dev
# or
yarn dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)를 열어 결과를 확인하세요.

## 사용 기술

- Next.js
- React
- TypeScript
- CSS
